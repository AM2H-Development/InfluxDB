# InfluxDB

## install:
* git clone https://github.com/AM2H-Development/InfluxDB.git
* cd InfluxDB
* cp sample.env .env
* nano .env # adopt Project and Hostname
* docker compose up -d

## data:
all data is stored in volume "influx_data" and "influx_conf"

## use:
* first time default login is: admin/admin
